# dice-game
2 Player dice game

Aim: To score 100 points before the other player

if dice ==1:
  current_score resets to 0
else:
  dice value is added to current score
  
dice roll button: 
  Rolls the dice

hold button:
  adds the current score to total score
  activates the second player
  
New game button:
  Starts a new game
